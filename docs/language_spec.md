# Спецификация языка MiniCompiler

## 1. Обзор

MiniCompiler — упрощённый C-подобный язык. Кодировка: **UTF-8** (BOM игнорируется).

---

## 2. Лексическая грамматика (EBNF)

```ebnf
program         ::= { token } ;
token           ::= whitespace | comment | keyword | identifier | literal | operator | delimiter ;
whitespace      ::= " " | "\t" | "\n" | "\r" ;
comment         ::= "//" { любой символ кроме newline } newline
                  | "/*" { любой символ } "*/" ;
```

---

## 3. Ключевые слова

```
fn, int, float, bool, string, if, else, while, for, return, true, false, void, struct
```

- **Регистрозависимые** (только строчные буквы)
- **Нельзя использовать как идентификаторы**

---

## 4. Идентификаторы

```ebnf
identifier ::= letter { letter | digit | "_" } ;
letter     ::= "a"..."z" | "A"..."Z" ;
digit      ::= "0"..."9" ;
```

**Правила:**
- Начинается с буквы или `_`
- Максимальная длина: **255 символов**
- Регистрозависимый

**Примеры:** `x`, `counter`, `_private`, `camelCase`

---

## 5. Литералы

### 5.1 Целые числа (Integer)
```ebnf
integer ::= ["-"] digit { digit } ;
```
- **Диапазон:** [-2147483648, 2147483647] (32-битное знаковое целое)
- **Примеры:** `42`, `-42`, `0`, `2147483647`

### 5.2 Числа с плавающей точкой (Float)
```ebnf
float ::= ["-"] digit { digit } "." digit { digit } ;
```
- **Примеры:** `3.14`, `0.5`, `-3.14`, `10.0`

### 5.3 Строки (String)
```ebnf
string ::= '"' { любой символ кроме '"' и '\n' } '"' ;
```
- **Примеры:** `"hello"`, `""`, `"Hello, World!"`, `"line1\nline2"`

### 5.4 Булевы значения (Boolean)
```
true, false
```

## 6. Операторы

| Категория | Операторы |

| **Логические** | `&&` (И), `\|\|` (ИЛИ), `!` (НЕ) |
| **Присваивание** | `=`, `+=`, `-=`, `*=`, `/=` |

## 8. Комментарии

| Тип | Синтаксис | Пример |
|-----|-----------|--------|
| **Однострочный** | `// ... до конца строки` | `// комментарий` |
| **Многострочный** | `/* ... */` (поддерживает вложенность) | `/* текст */` |

---

## 9. Обработка ошибок

| Ошибка | Сообщение |
|--------|-----------|
| Невалидный символ | `Ошибка в L:C: недопустимый символ 'X'` |
| Незакрытая строка | `Ошибка в L:C: незакрытый строковый литерал` |
| Незакрытый комментарий | Поглощается до EOF |
| Переполнение целого числа | `Ошибка в L:C: целочисленный литерал вне диапазона [-2^31, 2^31-1]` |
| Слишком длинный идентификатор | `Ошибка в L:C: идентификатор слишком длинный (максимум 255 символов)` |

**Формат ошибок:** Все сообщения об ошибках выводятся в формате `L:C ТИП_ТОКЕНА "сообщение"` или как часть токена `ILLEGAL`.
