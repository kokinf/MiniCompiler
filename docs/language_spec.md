# Спецификация языка MiniCompiler

## 1. Обзор

MiniCompiler — упрощённый C-подобный язык. Кодировка: **UTF-8** (BOM игнорируется).

---

## 2. Лексическая грамматика (EBNF)

```ebnf
program         ::= { token } ;
token           ::= whitespace | comment | keyword | identifier | literal | operator | delimiter ;
whitespace      ::= " " | "\t" | "\n" | "\r" ;
```

---

## 3. Ключевые слова

```
fn, int, float, bool, string, if, else, while, for, return, true, false, void, struct
```

- **Регистрозависимые** (только строчные буквы)
- **Нельзя использовать как идентификаторы**

---

## 4. Идентификаторы

```ebnf
identifier ::= letter { letter | digit | "_" } ;
letter     ::= "a"..."z" | "A"..."Z" ;
digit      ::= "0"..."9" ;
```

**Правила:**
- Начинается с буквы или `_`
- Максимальная длина: **255 символов**
- Регистрозависимый

**Примеры:** `x`, `counter`, `_private`, `camelCase`

---

## 5. Литералы

### 5.1 Целые числа (Integer)
```ebnf
integer ::= ["-"] digit { digit } ;
```
- **Диапазон:** [-2147483648, 2147483647]
- **Примеры:** `42`, `-42`, `0`

### 5.2 Числа с плавающей точкой (Float)
```ebnf
float ::= ["-"] digit { digit } "." digit { digit } ;
```
- **Примеры:** `3.14`, `0.5`, `-3.14`

### 5.3 Строки (String)
```ebnf
string ::= '"' { любой символ кроме '"' и newline } '"' ;
```
- **Примеры:** `"hello"`, `""`, `"Hello, World!"`

### 5.4 Булевы значения (Boolean)
```
true, false
```

---

## 6. Операторы

| Категория | Операторы |
|-----------|-----------|
| **Арифметические** | `+`, `-`, `*`, `/`, `%` |
| **Сравнения** | `==`, `!=`, `<`, `<=`, `>`, `>=` |
| **Логические** | `&&`, `||` |
| **Присваивание** | `=` |

---

## 7. Разделители

```
( ) { } [ ] ; , .
```

---

## 8. Комментарии

| Тип | Синтаксис | Пример |
|-----|-----------|--------|
| **Однострочный** | `// ... до конца строки` | `// комментарий` |
| **Многострочный** | `/* ... */` (поддерживает вложенность) | `/* текст */` |

---

## 9. Обработка ошибок

| Ошибка | Сообщение |
|--------|-----------|
| Невалидный символ | `invalid character 'X'` |
| Незакрытая строка | `unterminated string literal` |
| Незакрытый комментарий | Поглощается до EOF |
| Переполнение числа | `integer literal out of range [-2^31, 2^31-1]` |
| Длинный идентификатор | `identifier too long (max 255 characters)` |

**Формат:** `LINE:COLUMN ILLEGAL "Error at LINE:COLUMN: message"`
